# kotlin-blog

kotlin集成spring boot开发的一个博客站点，支持markdown写文章，文章列表分页、搜索查询等功能。

## 技术栈

- 编程语言：kotlin
- 数据库：mysql
- 框架：Spring Boot
- 模板引擎：Freemarker
- 前端组件库：JQuery、Bootstrap、Flat UI、Mditor、DataTables
- 构建工具：gradle

## 如何运行

- 下载代码到本地

  ```
  git clone https://github.com/zhenfeng-zhu/kotlin-blog.git
  ```

- 建表

  在mysql中，执行resources/sql/article.sql文件

- 使用idea打开项目

- 运行main函数即可
